Mail Log Messages

The default verbose logging level for @PACKAGE_NAME@ is to display all warnings and informational messages. Error messages are always reported. The debugging log messages are not explained. Each log message has a timestamp when the message was generated, the @PACKAGE_NAME@ process name and ID, the session ID of the current connected SMTP client, followed by the message number and text of the message.
